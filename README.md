# Data Analysis CLI Tool

Project ini adalah aplikasi berbasis Python untuk:
- Load data CSV
- Filtering data
- Statistik data
- Analisis sederhana
- Visualisasi data

## Fitur
- Filter umur
- Pencarian nama
- Statistik (mean, max, min)
- Analisis kota terbanyak
- Grafik distribusi umur
- Grafik jumlah per kota

## Tools
- Python
- Pandas
- Matplotlib
